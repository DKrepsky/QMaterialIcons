Detailed Description

Overview

The QMaterialIcon class provides the means to use Google's material icons inside a Qt project. It handles the resouces and all conversions to get icons and fonts inside your Qt application.

This class is a singleton, which means you don't have to instantiate an object to use it. Just add the qmaterialicons.h++ inside the files that need the icons and load the resources with QMaterialIcons::load() once. Also, don't forget to add the library to your project.

Warning
Make sure you load the library after the QApplication object instantiation.

Loading the resources

// You can load the resources inside you main.cpp or at the main window
// constructor.
#include "mainwindow.h++"
#include <QApplication>
#include "qmaterialicons.h++"
int main(int argc, char *argv[]) {
QApplication a(argc, argv); // <- make sure resources are loaded after the
// QApplication instantiation.
QMaterialIcons::load(); // <- The magic happens here.
MainWindow w;
w.show();
return a.exec();
}

Example - font

// This example shows how to set a label font with QMaterialIcons.
// First, grab the font object.
ui->label->setFont(QMaterialIcons::getFont());
// Set text to desired icon.
// Note that it is important to use a QChar to conver the wchar to text.
ui->label->setText(QChar(mi::heart));
// Use stylesheet to customize.
ui->label->setStyleSheet("color:red;font-size:30px;");
// Depending on the label dimentions, you should resize it.
ui->label->setFixedSize(30,30);

Example - icon

// This example shows how to set a push button icon.
// For icons, there is no need to use QChar.
ui->button->setIcon(QMaterialIcons::getIcon(mi::menu, Qt::black, 18));
